# 4. Overview / platform-manifest mechanism

- Status: seed
- Date: 2026-06-30
- Raised by: [ADR 0002](../decisions/0002-repo-landscape-poly-repo-principle.md)

## The thought

A deferred decision called for by ADR 0002. To prevent loss of overview across
many repos, we need a regenerated **platform manifest** listing every repo, its
taxonomy role, and version — derived rather than hand-maintained, conceptually the
same move decision-trail uses for `docs/overview.md`.

When this matures, promote it to its own ADR.

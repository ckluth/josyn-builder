# 5. Repo naming convention

- Status: seed
- Date: 2026-06-30
- Raised by: [ADR 0002](../decisions/0002-repo-landscape-poly-repo-principle.md)

## The thought

A deferred decision called for by ADR 0002. With 16+ repos, a predictable naming
scheme (e.g. `josyn-<concern>` / `josyn-<artifact>`) keeps the landscape scannable
and signals each repo's taxonomy role from its name alone.

When this matures, promote it to its own ADR.
